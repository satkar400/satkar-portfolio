## About the Project

The portfolio website is designed to present my work in a clean and professional way. It includes:

- **Home Section**: Introduction with a dynamic typing effect highlighting my roles.  
- **Profile Section**: Short biography and professional summary.  
- **Expertise / Skills Section**: Key skills in AI, ML, and software development.  
- **Portfolio Section**: Showcase of projects with links and descriptions.  
- **Contact Section**: Social media links and CV download option.  

The website is **responsive**, looks great on desktop and mobile, and is optimized for a modern, professional portfolio presentation.

---

## Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript  
- **Libraries**: FontAwesome for icons  
- **Features**: Animated typing effect, responsive layout, downloadable CV
